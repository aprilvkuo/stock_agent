## 🎯 系统能做什么？

### ✅ 1. 股票分析（核心功能）

你的系统可以分析**任何 A 股股票**，提供：

#### 📈 三大维度分析

| Agent | 分析内容 | 输出 |
|-------|----------|------|
| **基本面** | 财报数据、估值水平、财务健康 | 买入/卖出/持有 + 置信度 |
| **技术面** | K 线形态、技术指标、趋势判断 | 买入/卖出/持有 + 置信度 |
| **情绪面** | 市场舆情、热度分析、资金流向 | 乐观/中性/悲观 + 置信度 |

#### 🎯 综合投资建议

- **评级**: 🟢 推荐 / 🟡 观察 / 🔴 回避
- **仓位**: 0% - 15%（根据评级自动计算）
- **目标价**: 预期上涨空间
- **止损价**: 风险控制线

### ✅ 2. 预测验证（追踪准确率）

系统会自动记录预测，30 天后验证：

```
待验证预测示例:
┌─────────────────────────────────────┐
│ 股票    │ 预测内容    │ 验证日 │ 状态 │
├─────────────────────────────────────┤
│ 五粮液  │ 突破 ¥153   │ 04-07  │ ⏳   │
│ 五粮液  │ 不跌破 ¥138 │ 04-07  │ ⏳   │
│ 贵州茅台│ 突破 ¥1764  │ 04-07  │ ⏳   │
│ 贵州茅台│ 不跌破 ¥1596│ 04-07  │ ⏳   │
└─────────────────────────────────────┘
```

**自动验证**:
- 每日 09:00 检查到期预测
- 获取真实股价对比
- 记录准确率统计

### ✅ 3. 多 Agent 协作

```
5 个独立 Agent 协同工作:

┌──────────────────────────────────────┐
│  主 Agent (协调者)                    │
│  ├─ 接收请求                          │
│  ├─ 分发任务                          │
│  └─ 汇总决策                          │
└──────────────────────────────────────┘
           ↓
┌─────────┬─────────┬─────────┐
│基本面   │技术面   │情绪面   │
│Agent    │Agent    │Agent    │
└─────────┴─────────┴─────────┘
           ↓
    ┌──────────────┐
    │  复盘 Agent   │
    │ (定期验证)   │
    └──────────────┘
```
