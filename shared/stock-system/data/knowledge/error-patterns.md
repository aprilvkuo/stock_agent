## 📊 常见错误模式

### 1. 网络 API 失败

**错误现象**:
```
Remote end closed connection without response
```

**原因**:
- 腾讯财经 API 限流
- 网络波动
- API 服务临时不可用

**解决方案**:
1. 自动重试（3 次，间隔 2 秒）
2. 使用降级数据（缓存）
3. 记录失败时间，避开高峰

**代码位置**: `agent-fundamental.py:get_stock_data()`

---

### 2. 文件锁竞争

**错误现象**:
```
File "agent-fundamental.py", line 280
PermissionError / IOError
```

**原因**:
- 多个守护进程实例同时运行
- 同时写入同一个请求文件

**解决方案**:
1. PID 文件锁（单实例）
2. 请求级文件锁（防并发）
3. 死信队列（失败隔离）

**代码位置**: `daemon.py:check_single_instance()`

---

### 3. Python 版本冲突

**错误现象**:
- Python 3.9 和 3.14 混用
- 模块导入失败

**解决方案**:
1. 统一使用 `python3` 命令
2. 检查 shebang 行
3. 清理旧进程
