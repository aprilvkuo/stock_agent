# 进化报告集成文档

## 📊 功能概述

进化报告已完全集成到股票多 Agent 系统监控网站，你可以方便地查看系统的自我进化历史。

---

## 🎯 访问方式

### 1. 从首页进入
访问 http://localhost:5001

在页面顶部找到导航链接：
```
📊 股票多 Agent 系统监控
实时追踪分析进展 · 自动验证预测
最后更新：2026-03-08 XX:XX:XX  🤖 查看 Agent 详细状态  🧬 查看进化报告
```

点击 **🧬 查看进化报告** 即可进入。

### 2. 直接访问
```
http://localhost:5001/evolution
```

---

## 📄 页面功能

### 左侧：报告列表
- 显示所有进化报告 (按时间倒序)
- 每份报告显示：
  - 📅 执行时间
  - ✅ 成功次数
  - 📊 成功率
  - 📈 分析的报告数

### 右侧：报告详情
- 点击任意报告查看完整内容
- Markdown 格式渲染
- 包含：
  - 执行时间
  - 执行的任务
  - 日志记录
  - 统计数据

### 顶部统计卡片
- 📊 平均成功率
- ✅ 平均成功次数
- 📈 平均报告数

## 🔧 API 端点

### 1. 获取报告列表
```
GET /api/evolution
```

**响应示例**:
```json
{
  "reports": [
    {
      "filename": "evolution-20260308-014827.md",
      "time": "2026-03-08 01:48:27",
      "total_reports": 14,
      "success_count": 20,
      "success_rate": 100.0,
      "preview": "# 自我进化报告..."
    }
  ],
  "count": 1
}
```

### 2. 获取报告详情
```
GET /api/evolution/<filename>
```

**响应示例**:
```json
{
  "filename": "evolution-20260308-014827.md",
  "content": "# 自我进化报告\n\n## 执行时间\n...",
  "html": "# 自我进化报告\n..."
}
```

## ❓ 常见问题

### Q: 为什么看不到进化报告？
**A**: 系统需要运行一段时间才会生成进化报告。等待系统空闲 3 分钟或 1 小时定时触发。

### Q: 如何手动触发进化？
**A**: 运行 `python3 self-evolution.py` 即可立即生成进化报告。

### Q: 报告保存在哪里？
**A**: `/Users/egg/.openclaw/workspace/agents/stock-coordinator/data/evolution-*.md`

### Q: 可以删除旧报告吗？
**A**: 可以，建议保留最近 10-20 份报告即可。
