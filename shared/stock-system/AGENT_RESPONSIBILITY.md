## 🏆 奖惩执行流程

### 惩罚流程

```
问题发现 → 质检 Agent 记录 → 创建惩罚工单 → 执行惩罚 → 记录档案
   ↓
质检 Agent    质检 Agent     程序员 Agent    各 Agent    人事档案
```

**惩罚等级**:
| 等级 | 触发条件 | 惩罚措施 |
|------|---------|---------|
| ⚠️ 警告 | 轻微违规 | 记录 + 口头警告 |
| 🟡 轻度 | 重复违规 | 权重 -0.05，暂停工作 1 小时 |
| 🟠 中度 | 严重失误 | 权重 -0.10，暂停工作 24 小时，强制学习 |
| 🔴 严重 | 重大过失 | 权重 -0.20，停职 7 天，重新培训 |
| ⚫ 开除 | 屡教不改 | 从系统中移除（需人类确认） |

### 奖励流程

```
成就达成 → 质检 Agent 验证 → 创建奖励工单 → 执行奖励 → 荣誉榜
   ↓
各 Agent      质检 Agent      主 Agent      各 Agent    公示
```

**奖励类型**:
| 类型 | 奖励内容 | 有效期 |
|------|---------|--------|
| 权重提升 | 决策权重 +0.05~0.20 | 永久/临时 |
| 荣誉榜 | 月度优秀 Agent | 公示 30 天 |
| 特别奖励 | 重大贡献额外奖励 | 一次性 |
| 晋升 | 升级 Agent 版本 | 永久 |

## ⚖️ 申诉机制

### Agent 申诉流程

```
惩罚决定 → Agent 不服 → 提交申诉 → 质检复核 → 主 Agent 仲裁 → 最终决定
   ↓
质检 Agent   被惩罚 Agent   申诉工单    质检 Agent    主 Agent     执行
```

**申诉条件**:
- 有证据证明不是本 Agent 责任
- 惩罚明显不公平
- 外部因素导致问题

**申诉时限**: 惩罚后 1 小时内
