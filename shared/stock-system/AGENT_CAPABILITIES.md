### 4. 🎯 主 Agent（Coordinator）

**职责**: 汇总各 Agent 判断，给出综合评级

**核心能力**:
```python
- check_results(request_id)   # 检查结果是否完成
- get_stock_data(stock_code)  # 获取股价数据
- AgentCollaboration          # Agent 协作流程
- process_pending()           # 处理待汇总请求
```

**工作流程**:
1. 等待基本面/技术面/情绪分析完成
2. 启动 Agent 协作（各 Agent 注册初始判断）
3. 共享上下文，允许 Agent 调整判断
4. 动态权重计算（根据历史准确率）
5. 生成综合评级和目标价
6. 写入验证队列

**输出**:
- 综合评级：强烈买入/买入/持有/卖出/强烈卖出
- 置信度：0-100%
- 目标价：基于 PE/技术位
- 风险提示

**今日工作**: 424 次  
**触发条件**: 事件驱动（有完成的分析才工作）
