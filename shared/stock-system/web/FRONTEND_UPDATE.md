## 🆕 新增功能

### 1. v2.0 分析报告列表

**路径**: `/v2-reports`

**功能**:
- ✅ 显示所有 v2.0 分析报告
- ✅ 按时间排序（最新在前）
- ✅ 评级标签（强烈买入/买入/持有/卖出）
- ✅ 点击查看详情

### 2. 验证队列

**功能**:
- ✅ 显示所有待验证预测
- ✅ 显示目标价和止损价
- ✅ 显示验证日期和状态

### 3. API 端点

| 端点 | 功能 |
|------|------|
| `/api/v2/logs` | 获取分析日志列表 |
| `/api/v2/log/<filepath>` | 获取单个报告详情 |
| `/api/v2/validation` | 获取验证队列 |

## 🚀 使用流程

### 1. 分析股票

```bash
cd /Users/egg/.openclaw/workspace/memory/stock-system

# 分析单只股票
python3 scripts/auto_agent_v2.py analyze 600519

# 分析核心股票池
python3 scripts/analyze_core_pool.py
```

### 2. 查看报告

打开浏览器访问：
```
http://localhost:5001/v2-reports
```

### 3. 查看详情

点击报告列表中的 "📄 查看详情" 按钮

## 🐛 故障排查

### Q: 页面显示 404？
A: 确保 Web 服务正在运行

### Q: API 返回空列表？
A: 检查 `memory/stock-system/analysis-log/` 目录是否有文件

### Q: 页面样式不正常？
A: 清除浏览器缓存，强制刷新（Cmd+Shift+R）
