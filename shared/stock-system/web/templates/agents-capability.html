🤖 Agent 能力说明

了解每个 Agent 的职责、数据源和分析能力

📊

基本面 Agent

Fundamental Analysis

🎯 核心职责

  • • 分析公司财务报表
  • • 评估估值水平 (PE/PB/ROE)
  • • 判断财务健康状况
  • • 分析营收和利润增长

📚 数据来源

东方财富 API 财报数据 F10 资料

📈 分析指标

PE 市盈率
PB 市净率
ROE 净资产收益率
营收增速
利润增速
毛利率
净利率
负债率

⚙️ 评估标准

  • ✅ ROE > 25%: 优秀
  • ✅ PE < 15: 低估
  • ✅ 营收增速 > 20%: 高增长
  • ⚠️ PB > 8: 偏高
📈

技术面 Agent

Technical Analysis

🎯 核心职责

  • • 分析 K 线走势
  • • 计算技术指标
  • • 判断支撑/阻力位
  • • 识别买卖信号

📚 数据来源

东方财富 API K 线数据 成交量

📈 技术指标

MA5/20/60 均线
RSI 相对强弱
MACD 平滑异同
KDJ 随机指标
BOLL 布林带
成交量
支撑位
阻力位

⚙️ 判断标准

  • ✅ 均线多头排列：看涨
  • ✅ MACD 金叉：买入信号
  • ✅ RSI 30-70: 正常区间
  • ⚠️ RSI > 70: 超买
😊

情绪 Agent

Sentiment Analysis

🎯 核心职责

  • • 分析市场舆情
  • • 监控涨跌幅
  • • 分析成交量异动
  • • 评估市场热度

📚 数据来源

东方财富 API 新闻资讯 股吧评论

📈 分析维度

涨跌幅
成交量
资金流向
新闻舆情
机构评级
社交媒体
龙虎榜
大宗交易

⚙️ 判断标准

  • ✅ 涨幅 > 3%: 强势
  • ✅ 成交量放大：关注
  • ✅ 正面新闻多：乐观
  • ⚠️ 大幅波动：谨慎
🎯

主 Agent

Coordinator

🎯 核心职责

  • • 汇总三个 Agent 结果
  • • 综合评估投资建议
  • • 生成最终评级
  • • 写入验证队列

📚 数据来源

基本面报告 技术面报告 情绪面报告

⚙️ 决策逻辑

  • • 三个维度综合评分
  • • 权重：基本面 40% + 技术面 30% + 情绪面 30%
  • • 生成：推荐/观望/回避
  • • 给出目标价和止损位
📝

复盘 Agent

Review Agent

🎯 核心职责

  • • 验证历史预测准确性
  • • 分析预测偏差原因
  • • 优化分析模型
  • • 生成改进建议

📚 数据来源

历史预测 实际走势

⚙️ 工作流程

  • • 定期验证预测结果
  • • 统计成功率
  • • 识别错误模式
  • • 更新知识库
📝

复盘 Agent（增强版）

Review Agent

🎯 核心职责

  • • 验证预测准确性

  • • 更新知识库和经验规则
  • • 生成改进建议和复盘报告

⏰ 触发条件

🔍 如何查看 Agent 工作细节

📄 查看分析结果文件

• 位置：/agents/stock-coordinator/data/queue/results/
• 文件名：result-REQUEST-ID-fundamental.md
• 包含：数据来源、分析过程、判断依据

📊 在报表网站查看

• 访问：http://localhost:5001/report/股票代码
• 查看：三个维度的详细分析
• 包含：数据指标、判断理由、风险提示

📊 查看复盘报告

• 访问：http://localhost:5001/evolution
（复盘报告页面）
• 查看：Agent 性能统计
• 包含：成功率、改进计划

📋 查看系统日志

• 位置：/agents/stock-coordinator/data/logs/
• 文件：daemon-YYYY-MM-DD.log
• 包含：每个 Agent 的执行记录
