## 🎯 功能特性

### 核心功能
- ✅ **多市场支持**: A 股、港股、美股
- ✅ **持仓管理**: 添加、编辑、删除持仓
- ✅ **实时汇总**: 按市场统计持仓数量
- ✅ **图片上传**: 支持拖拽上传（OCR 功能预留）
- ✅ **响应式设计**: 适配手机、平板、电脑

### 数据字段
| 字段 | 说明 | 示例 |
|------|------|------|
| 股票代码 | 唯一标识 | 600519 / 00700 / AAPL |
| 股票名称 | 显示名称 | 贵州茅台 / 腾讯控股 / 苹果 |
| 股数 | 持仓数量 | 100 |
| 成本价 | 买入成本 | 1500.00 |
| 市场类型 | A/HK/US | A |
| 备注 | 可选说明 | 长期持有、定投等 |

## 🚀 使用方法

### 网页访问

1. **打开持仓管理页面**
   ```
   http://localhost:5000/holdings
   ```

2. **添加持仓**
   - 输入股票代码（自动识别市场）
   - 填写股票名称、股数、成本价
   - 可选添加备注
   - 点击"添加持仓"

3. **管理持仓**
   - 编辑：点击"✏️ 编辑"修改信息
   - 删除：点击"🗑️ 删除"移除持仓

### API 使用

#### 获取持仓列表
```bash
GET /api/holdings?status=active
```

响应:
```json
{
  "holdings": [
    {
      "id": "20260308125047_600519",
      "stock_code": "600519",
      "stock_name": "贵州茅台",
      "shares": 100,
      "cost_price": 1500.0,
      "market_type": "A",
      "notes": "长期持有"
    }
  ],
  "summary": {
    "total_count": 1,
    "by_market": {"A": 1, "HK": 0, "US": 0},
    "total_cost": 150000.0
  }
}
```

#### 添加持仓
```bash
POST /api/holdings
Content-Type: application/json

{
  "stock_code": "600519",
  "stock_name": "贵州茅台",
  "shares": 100,
  "cost_price": 1500.0,
  "market_type": "A",
  "notes": "长期持有"
}
```

#### 更新持仓
```bash
PUT /api/holdings/{holding_id}
Content-Type: application/json

{
  "shares": 200,
  "cost_price": 1450.0
}
```

#### 删除持仓
```bash
DELETE /api/holdings/{holding_id}
```

## 📋 股票代码格式

### A 股
- **格式**: 6 位数字
- **示例**: 600519（贵州茅台）、000858（五粮液）、300750（宁德时代）
- **规则**: 
  - 60xxxx - 上交所
  - 00xxxx - 深交所主板
  - 30xxxx - 深交所创业板

### 港股
- **格式**: 5 位数字
- **示例**: 00700（腾讯控股）、09988（阿里巴巴）
- **规则**: 全部以 0 开头

### 美股
- **格式**: 1-5 位字母
- **示例**: AAPL（苹果）、TSLA（特斯拉）、BABA（阿里巴巴）
- **规则**: 大写字母组合
