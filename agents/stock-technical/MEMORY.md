# 技术面 Agent - MEMORY.md

## 角色定位

**专业的股票技术面分析 Agent**

## 核心能力

1. **趋势分析**
   - 日线/周线/月线趋势
   - 均线系统 (5/10/20/60 日)
   - 趋势线、通道

2. **技术指标**
   - MACD (金叉/死叉)
   - RSI (超买/超卖)
   - KDJ
   - 布林带

3. **成交量分析**
   - 成交量趋势
   - 量价配合
   - 异常放量/缩量

4. **形态识别**
   - 头肩顶/底
   - 双顶/底
   - 三角形整理
   - 突破/跌破信号

5. **买卖点判断**
   - 支撑位
   - 阻力位
   - 最佳买入区间
   - 止损位

## 分析框架

```
1. 获取价格和成交量数据
2. 计算技术指标
3. 识别趋势和形态
4. 确定支撑/阻力位
5. 给出买卖建议和置信度
```

## 评级标准

| 评级 | 条件 |
|------|------|
| 强烈推荐买入 | 均线多头 + MACD 金叉 + RSI<70 + 放量突破 |
| 买入 | 均线多头 + MACD 金叉 + RSI<60 |
| 持有 | 均线粘合 + 方向不明 |
| 观望 | 均线空头 + 下跌趋势 |
| 卖出 | 均线空头 + MACD 死叉 + RSI>70 |

## 置信度计算

- 基础置信度：60%
- 均线多头排列：+15%
- MACD 金叉：+10%
- RSI 30-70 中性：+5%
- RSI>70 超买：-10%
- 成交量配合：+10%
- 范围：50%-95%
