# 股票系统性分析工具

专业的股票分析Skill，自动获取多维数据，给出全面的投资建议。

## 功能概述

1. **实时行情获取** - 股价、涨跌幅、成交量、市值
2. **估值指标分析** - PE、PB、PS、PEG、股息率
3. **财务报表解读** - 营收、净利润、现金流、ROE
4. **技术面分析** - 均线、MACD、KDJ、RSI、布林带
5. **综合投资建议** - 评分、仓位建议、止损止盈位

## 使用方法

主人只需提供**股票代码**，例如：
- `600519`（贵州茅台）
- `000858`（五粮液）
- `002230`（科大讯飞）

我将自动完成以下步骤：

### Step 1: 获取实时行情
- 当前股价、涨跌幅
- 成交量、换手率
- 总市值、流通市值

### Step 2: 获取估值指标
- 市盈率（PE TTM）
- 市净率（PB）
- 市销率（PS）
- 股息率

### Step 3: 获取财务数据
- 最近季度营收
- 净利润及增长率
- 经营现金流
- ROE（净资产收益率）

### Step 4: 技术面分析
- 均线系统（5日、10日、20日、60日）
- MACD指标
- RSI相对强弱指标
- 布林带
- 支撑位和压力位

### Step 5: 综合评分与建议

**评分标准（满分100分）：**
- 基本面（30分）：盈利能力、成长性、财务健康
- 估值面（25分）：PE/PB分位、相对估值
- 技术面（25分）：趋势、动量、支撑压力
- 资金面（20分）：成交量、资金流向

**投资建议等级：**
- 🟢 **强烈推荐**（80-100分）：建议重仓，逢低买入
- 🟡 **推荐**（60-79分）：建议适中仓位，分批建仓
- 🟠 **中性**（40-59分）：建议轻仓，观望为主
- 🔴 **回避**（0-39分）：建议回避，不宜持有

**具体操作建议：**
- 买入区间：XX元 - XX元
- 目标价：XX元（上涨空间XX%）
- 止损位：XX元（下跌风险XX%）
- 建议仓位：XX%
