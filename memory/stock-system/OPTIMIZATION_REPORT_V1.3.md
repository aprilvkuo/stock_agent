## 📊 技术细节

### 1. 腾讯财经 API

**API 地址**: `http://qt.gtimg.cn/q={代码}`

**支持格式**:
- A 股上海：`sh600519`
- A 股深圳：`sz000858`
- 港股：`hk00700`

**返回格式**: GBK 编码的 JSONP

**示例响应**:
```
v_sh600519="51~贵州茅台~600519~1402.00~1361.00~..."
```

**解析字段**（57 个）:
| 字段 | 索引 | 说明 |
|------|------|------|
| 股票名称 | 1 | 贵州茅台 |
| 当前价 | 3 | 1402.00 |
| 昨收价 | 4 | 1361.00 |
| 涨跌幅 | 20 | +3.00% |
| 成交量 | 23 | 1403 手 |
| 总市值 | 25 | 1403.96 亿元 |
| PE(TTM) | 27 | 1403.97 |
| PB | 28 | 2.0 |
| ROE | 52 | 20.37% |

**核心函数**:
```python
# 获取完整数据
def get_stock_data_full(stock_code) -> Dict:
    # 1. 调用腾讯 API 获取实时行情
    realtime_data = fetch_tencent_stock_data(stock_code)
    
    # 2. 获取财务数据（缓存）
    financial_data = get_financial_data(stock_code)
    
    # 3. 获取技术指标（缓存）
    technical_data = get_technical_indicators(stock_code)
    
    # 4. 合并返回
    return merge_data(realtime, financial, technical)
```

**降级策略**:
```python
def get_stock_data(stock_code):
    if TENCENT_API_AVAILABLE:
        data = get_stock_data_full(stock_code)
        if data and data.get('price'):
            return data  # 使用真实数据
    
    # API 失败时降级到模拟数据
    return get_mock_stock_data_fallback(stock_code)
```
