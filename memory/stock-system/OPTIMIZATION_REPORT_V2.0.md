### 2. 风险评估师 (risk_assessor.py)

**路径**: `/Users/egg/.openclaw/workspace/memory/stock-system/scripts/risk_assessor.py`

**核心职责**:
1. 综合评估股票风险水平
2. 计算建议仓位上限
3. 设定止损位
4. 识别风险因素

**分析维度**:
| 维度 | 权重 | 评估内容 |
|------|------|----------|
| 估值风险 | 25 分 | PE/PB 偏高程度 |
| 基本面风险 | 25 分 | ROE/营收/利润增长 |
| 技术面风险 | 25 分 | 均线排列/RSI 超买 |
| 情绪面风险 | 25 分 | 情绪过热/背离 |

**输出示例**:
```json
{
  "risk_level": "low",
  "risk_score": 8/100,
  "position_limit": 0.30,
  "stop_loss_price": 1261.80,
  "risk_factors": ["无明显风险因素"]
}
```

---

### 3. CIO 决策模块 (cio_decision.py)

**路径**: `/Users/egg/.openclaw/workspace/memory/stock-system/scripts/cio_decision.py`

**核心职责**:
1. 汇总 4 个 Agent 分析结果
2. 根据风险调整仓位
3. 生成最终 BUY/HOLD/SELL 建议
4. 输出完整可执行的投资建议

**决策流程**:
```
1. 计算各 Agent 评分 → 加权平均
2. 根据风险等级调整评分
3. 确定行动 (STRONG_BUY/BUY/HOLD/SELL)
4. 根据风险限制仓位
5. 计算目标价和止损价
6. 生成完整报告
```

**输出示例**:
```json
{
  "action": "STRONG_BUY",
  "action_cn": "强烈买入",
  "confidence": 0.73,
  "position": 0.25,
  "target_price": 1542.20,
  "stop_loss": 1261.80,
  "time_horizon": "3-6 个月"
}
```
