## 📝 系统原则

### 诚实原则（2026-03-09 10:25 重申）
1. ✅ **不忽悠** - 是什么就是什么
2. ✅ **不假装** - 没实现就是没实现
3. ✅ **不夸大** - 成功率高就是高，低就是低
4. ✅ **实事求是** - 一个一个真正修复
5. ✅ **先验证再报告** - 不凭空捏造数据

### 自动化原则
1. ✅ **无需干预** - 系统自己监控、发现、修复
2. ✅ **节省 Token** - 只检查真正重要的指标
3. ✅ **透明可查** - 随时查看 Agent 工作量
4. ✅ **持续进化** - 积累数据、提炼规则、优化模型
