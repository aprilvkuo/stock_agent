## 🏗️ 架构重构 (v6.0)

### 优化前
```
auto_agent.py (单进程)
├── analyze_fundamental()
├── analyze_technical()
└── analyze_sentiment()
```
**问题**: 记忆共享、难以调试、无法扩展

### 优化后
```
main_coordinator.py (协调者)
    ├── fundamental_agent.py (独立进程)
    ├── technical_agent.py (独立进程)
    └── sentiment_agent.py (独立进程)
```
**优势**: 记忆隔离、并发执行、易于扩展
